# sandbox-rn

## Overview

[React Native](https://reactnative.dev/) Sandbox, created with `npx react-native init`.

### Getting Started

```bash
nvm use
yarn install
bundle install
cd ios
bundle exec pod install
```

### Android

Have an Android emulator running.

```bash
npx react-native run-android
```

### iOS

```bash
npx react-native run-ios
```

### Testing

```bash
npm run test
```
